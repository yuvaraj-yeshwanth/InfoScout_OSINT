
# InfoScout

InfoScout is a Python-based OSINT tool for information gathering.

## Features
- IP Tracker
- Phone Number Tracker
- MAC Address Lookup
- Email Scanner
- Metadata Extractor
- Reverse Image Search
- Port Scanner
- CMS Detection
- NSLookup
- Reverse IP Lookup
- Subdomain Enumeration
- Whois Lookup
